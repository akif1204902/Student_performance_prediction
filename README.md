# Student Performance Prediction

This project predicts student final scores based on academic factors using machine learning.

## Problem Type
Regression

## Features
- Study hours
- Attendance
- Previous scores

## Algorithm Used
- Linear Regression

## Libraries
- Pandas
- Scikit-learn

## Workflow
1. Load dataset
2. Split data into training and testing sets
3. Train regression model
4. Evaluate using MAE and RÂ² score

1. Install requirements  
   pip install -r requirements.txt
2. Run the program  
   python student_performance.py
